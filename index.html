SID - C64 sound chip

OSAMC 3/2017

What is it?


  • 3 channel digital controlled synth in a single chip
  • One chip in each C64 (1982)
  • 10 - 17 million units sold
  • used heavily for games, intros, demos

How does it sound?


  • Because Nils doesn't like theoretical talks
  • Youtube link:Katakis soundtrack (originally shown with jsSID)

Features


  • 4 Waveforms types: Triangle, Sawtooth, Square, Noise
  • a ADSR waveform envelope generator for each channel
  • 1 filter for all channels
  • Ring modulation of channels possible
  • playing 4 bit samples is possible through tricks but cpu-time-consuming

Hardware


  • Multiple Revisions MOS 6581/6582/8580
  • First version sound different due to different filter design
  • Designed by Robert Yannes, who later build the Ensoniq SQ1

Registers


  • 29 8bit registers
  • 7 registers per voice: frequency, pulse width, control, ADSR
  • 4 global registers: filter cutoff, resonance, filter mode, volume
  • Read-only registers for A/D conversion

Composing restrictions


  • 3 channels
  • Memory
  • register control time to set the registers - depending on context as low as 63 cpu cycles (one raster line)
  • Tooling requires technical knowledge (eg. hex numbers, layout and function of SID registers)

Free software for composing


  • most editors have a tracker style interface
  • SIDRip - a C64 SID-Editor - usable on emulators
  • Cheesecutter - multi platform GPL editor
  • GoatTracker - another multiplatform GPL editor. newer versions can handle midi input.
  • + lots of native editors with various licenses
